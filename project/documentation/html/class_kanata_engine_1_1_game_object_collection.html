Detailed Description

Provides a collection to manage game objects and their lifecycle.

The GameObjectCollection class is a static class that manages game objects and their lifecycle. It allows adding, updating, rendering, and disposing of game objects. The class provides functionalities to find game objects by name and inspect the active game objects.
