Detailed Description

Contains classes related to the splash art scene of the application.

The SplashArtScene namespace encapsulates classes specific to the splash art scene of the application. It defines a SplashScene class that inherits from BaseScene and showcases loading behavior, input handling, rendering, and managing game objects and their behaviors related to the splash art display.
