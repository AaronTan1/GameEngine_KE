Detailed Description

Contains classes and utilities for the KanataEngine application.

The KanataEngine namespace encapsulates various classes and utilities that contribute to the KanataEngine application. It includes a ProgressBarComponent class that represents a progress bar component for visual feedback on progress. The class provides methods to set the initial position, progress value, target progress, bar width, and size.
