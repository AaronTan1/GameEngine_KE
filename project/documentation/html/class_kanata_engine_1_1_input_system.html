Detailed Description

Contains classes and utilities for the KanataEngine application.

The KanataEngine namespace encapsulates various classes and utilities that contribute to the KanataEngine application. It includes an InputSystem class to handle input processing and provides functions to query key and mouse button states. Additionally, it defines a Vector2 struct to represent 2D coordinates.
