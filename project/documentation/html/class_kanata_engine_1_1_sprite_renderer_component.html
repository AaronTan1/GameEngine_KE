Detailed Description

Contains classes and utilities for the KanataEngine application.

The KanataEngine namespace encapsulates various classes and utilities that contribute to the KanataEngine application. It includes a SpriteRendererComponent class that represents a component for rendering sprites. The class provides methods to draw sprites, set colors, opacity, size, and pivot points. Additionally, it defines Vec2 and Vec3 structs to represent 2D coordinates and RGB colors.
