Detailed Description

Contains classes related to the loading scene of the application.

The LoadingScene namespace encapsulates classes specific to the loading scene of the application. It defines a LoadGameScene class that inherits from BaseScene and showcases loading behavior, input handling, rendering, and managing game objects and their behaviors related to the loading process.
