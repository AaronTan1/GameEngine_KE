Detailed Description

Contains classes and utilities for the KanataEngine application.

The KanataEngine namespace encapsulates various classes and utilities that contribute to the KanataEngine application. It includes a RenderComponent class that represents a rendering component for managing visual rendering properties. The class provides methods to set the color (RGB) of the rendered object and interacts with TransformComponent for transformations.
